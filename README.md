# Hashcode 2020 Practice Round 

This repository aims to find a solution for the hashcode 2020 practice round. Instructions for this problem can be found under the `problem` directory.

To use this code, do the following:

```bash
python3 -m virtualenv .venv --python=/usr/bin/python3
.venv/bin/pip install -r requirements.txt
.venv/bin/python main.py
```
